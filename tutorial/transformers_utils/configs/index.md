# configs 子模块索引

## 模块概述

`configs/` 子模块为 vllm-omni 支持的自定义模型提供 HuggingFace Transformers 兼容的配置类注册，使这些模型能被 `AutoConfig` 自动识别和加载。

## 文档列表

| 文件 | 说明 |
|------|------|
| [__init__.md](__init__.md) | 包初始化与配置类注册 |
| [fish_speech.md](fish_speech.md) | Fish Speech 模型配置 |
| [mammoth_moda2.md](mammoth_moda2.md) | Mammoth-Moda2 模型配置 |

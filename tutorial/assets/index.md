# assets 模块教程 — 资源工具

## 模块概述

`assets/` 模块提供多媒体资源处理的工具函数，当前主要包含视频音频提取功能。

## 架构图

```
assets/
└── video.py          # 视频音频提取工具
        │
        ▼
   ┌──────────┐
   │  librosa  │  ← 音频处理库
   └──────────┘
        │
        ▼
   numpy.ndarray      ← 音频信号输出
```
